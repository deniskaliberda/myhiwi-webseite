# Project Requirements: MyHiwi

## Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Components:** Radix UI oder Shadcn/ui (für saubere Barrierefreiheit)

## SEO Requirements
- Semantisches HTML (H1 pro Seite, korrekte Sektions-Tags)
- Image Alt-Texte sind Pflicht
- Mobile-First Response Design
- Schnelle Ladezeiten (Core Web Vitals optimiert)

## Navigation Structure
- Home
- Leistungen (Local SEO, Webdesign)
- Preise
- Über mich (Nationalspieler Story)
- Kontakt (Calendly Link)
